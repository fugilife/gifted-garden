# 🌱 吉福種樹花園 · Gifted Garden

專為 Gisele（8歲）與 Fiona（6歲）設計的中文學習激勵遊戲

## ✨ 功能特色

- 🍎 **Gisele 的蘋果樹** - 從種子到豐收，5 個生長階段
- 🍊 **Fiona 的橘子樹** - 完整的成長視覺回饋
- 💰 **零用錢系統** - 每次完成中文練習賺 60 元
- 🪺 **鳥巢生態系統** - 累積 600 元解鎖，三隻彩色成鳥與蛋孵化
- 🦦🐰 **樹下動物** - 狐獴與兔子，隨著累積金額逐步出現
- 🐒 **猴子提領動畫** - 每次提領都有趣味拔果實動畫
- 🎉 **慶祝特效** - 完成動作時的視覺獎勵

## 🚀 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 在瀏覽器打開 http://localhost:3000
```

## 📦 部署

```bash
# 建置靜態網站
npm run build

# 自動部署至 Vercel
```

## 🎮 使用方式

1. 開啟網站，選擇 Gisele 或 Fiona 的花園
2. 點擊「我今天寫完中文字了！✏️」
3. 選擇澆水 💧 或施肥 🌿
4. 看著植物成長、零用錢增加！
5. 累積足夠金額後可提領零用錢

## 📊 解鎖時間軸

- **180 元** - 植物進入第 2 階
- **300 元** - 第 1 隻樹下動物出現
- **360 元** - 植物進入第 3 階
- **600 元** - 🪺 鳥巢降落！三隻成鳥到來
- **900 元** - 植物進入第 5 階（豐收）
- **1440 元** - 三隻雛鳥全孵化
- **1500 元** - 第 5 隻動物出現（最終狀態）

## 🛠️ 技術棧

- **Next.js 16** - React 框架
- **TypeScript** - 型別安全
- **Tailwind CSS** - 樣式系統
- **localStorage** - 資料持久化
- **純 CSS 動畫** - 所有動畫效果
