# 吉福種樹花園 · Gifted Garden
> 專為 Gisele（8歲）與 Fiona（6歲）設計的中文學習激勵遊戲

## 專案目標

孩子們每完成一次繁體中文字書寫練習，就可以打開這個網頁，為自己的植物**澆水**或**施肥**，讓植物生長，同時累積 60 元零用錢。目標是透過即時視覺回饋與成就感，持續激勵學習中文的動力。

## 核心玩法

1. 孩子選擇自己的角色（Gisele 或 Fiona）
2. 進入個人花園頁面，看到自己的植物
3. 點擊「我今天寫完中文字了！✏️」
4. 選擇澆水 💧 或 施肥 🌿（兩者都能使植物生長）
5. 植物執行生長動畫，畫面出現慶祝效果
6. 零用錢 +60 元，累積顯示
7. 植物根據累積次數呈現不同生長階段

## 植物設定

| 孩子 | 年齡 | 植物 | 生長主題 |
|------|------|------|---------|
| Gisele | 8 歲 | 🍎 蘋果樹 | 從幼苗 → 小樹 → 大樹 → 開花 → 結蘋果 |
| Fiona  | 6 歲 | 🍊 橘子樹 | 從幼苗 → 小樹 → 大樹 → 花苞 → 結橘子 |

> ⚠️ Fiona 的植物為**橘子樹**（非櫻花樹）。橘子樹會結果實，讓提領金額時的「拔果實」動畫更有意義。

## 生長階段（各植物 5 階段）

每 3 次澆水/施肥進升一階（前期較快，後期較慢）：
- 第 1 階：種子發芽 🌱（0–2 次）
- 第 2 階：幼苗 🌿（3–5 次）
- 第 3 階：小樹 🌳（6–9 次）
- 第 4 階：大樹開花 🌼（10–14 次）
- 第 5 階：豐收結果 🍎/🍊（15 次以上）

## 鳥巢生態系統（解鎖：累積 600 元）

**解鎖條件**：累積零用錢達到 **600 元**（即完成 10 次練習）

| 元素 | 描述 |
|------|------|
| 🪺 鳥巢 | 以 drop-in 動畫降落至樹枝，內有 3 顆蛋 |
| 🥚→🐥 孵化 | 每累積 120 元，一顆蛋升階（裂縫→破殼→雛鳥），共 3 顆 |
| 🐦 成鳥 A（餵食鳥） | 守在巢旁，每 4 秒啄食餵雛鳥，嘴含小蟲 |
| 🐦 成鳥 B（守護鳥） | 在樹旁來回飛翔守護，偶爾停棲 |
| 🐦 成鳥 C（歌唱鳥） | 停在較高樹枝，偶爾張嘴唱歌（音符飄出），偶爾飛走再回來 |

**三隻成鳥各為不同顏色**（詳見 DESIGN.md 成鳥配色）

解鎖瞬間有特別動畫與提示：「🪺 有鳥兒來這棵樹築巢了！因為這棵樹長得夠大了 ✨」

## 樹下動物系統（解鎖：每 300 元出現一隻，最多 5 隻）

| 孩子 | 動物 | 行為 |
|------|------|------|
| Gisele 🍎 | 🦦 狐獴（Meerkat） | 直立警戒、左右奔跑、偶爾挖地 |
| Fiona 🍊 | 🐰 兔子 | 原地跳躍、左右蹦跳、偶爾豎耳靜止 |

出現時機（以 totalMoney 計算）：
- 300 元：第 1 隻出現（從樹後探出頭）
- 600 元：第 2 隻出現
- 900 元：第 3 隻出現
- 1200 元：第 4 隻出現
- 1500 元：第 5 隻出現（最多 5 隻同時在場）

每隻動物有**獨立動畫循環**，彼此動作錯開，營造熱鬧感。

## 提領零用錢功能

- **入口**：花園頁面零用錢區塊旁的「提領 💸」按鈕
- **操作**：自由輸入金額（不可超過現有餘額）
- **保護**：無需密碼，孩子可自行提領
- **動畫**：每次提領都觸發猴子拔果實動畫
  - Gisele 花園：猴子從蘋果樹上摘走一顆蘋果 🐒🍎
  - Fiona 花園：猴子從橘子樹上摘走一顆橘子 🐒🍊
- **扣款後**：餘額即時更新，數字以「減少跳動」動畫呈現

## 技術規格

- **框架**：Next.js 14（App Router）
- **樣式**：Tailwind CSS
- **資料儲存**：localStorage（無需後端、無需登入）
- **部署**：Vercel（手機可直接打開網址）
- **語言**：繁體中文介面
- **響應式**：Mobile-first，針對手機優化

## 資料結構（localStorage）

```json
{
  "gisele": {
    "totalActions": 12,
    "totalMoney": 720,
    "availableMoney": 660,
    "lastAction": "2026-06-20",
    "history": [
      { "date": "2026-06-20", "type": "water", "moneyEarned": 60 }
    ],
    "withdrawals": [
      { "date": "2026-06-18", "amount": 60, "note": "" }
    ]
  },
  "fiona": {
    "totalActions": 8,
    "totalMoney": 480,
    "availableMoney": 480,
    "lastAction": "2026-06-20",
    "history": [],
    "withdrawals": []
  }
}
```

> `totalMoney` = 累計賺到的總金額（只增不減，決定樹的生長階段與所有解鎖條件）
> `availableMoney` = 實際可提領餘額（賺入 − 提領）
> **所有解鎖系統（植物階段、鳥巢、樹下動物）均以 `totalMoney` 計算，提領不影響成就**

## 花園頁面元素

1. **頂部**：孩子名字 + 累積零用錢 + 可提領餘額 + 提領按鈕
2. **中央**：植物 SVG 插圖（根據階段切換），含鳥巢/成鳥/樹下動物覆蓋層
3. **生長條**：目前在第幾階、距下一階還需幾次
4. **行動按鈕**：「我今天寫完中文字了！✏️」觸發 → 澆水/施肥選擇
5. **慶祝動畫**：星星、果實、金幣飛散效果
6. **提領動畫**：猴子出場拔果實 overlay
