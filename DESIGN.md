# DESIGN.md · 吉福種樹花園視覺設計系統

## 設計方向

**目標受眾**：6歲與8歲的孩子，在菲律賓的台灣家庭，自學繁體中文
**核心情感**：驚喜感、成就感、溫暖歸屬感
**美學定位**：溫暖水彩童書風格 × 台灣傳統節慶色彩，有別於一般遊戲 app 的塑膠感

## 植物 SVG 設計規範

每個植物有 5 個 SVG 狀態，設計要點：
- 線條圓潤，無銳利角
- 水彩質感（用半透明漸層模擬）
- 尺寸：280×320px viewBox
- 動畫：持續輕微上下漂浮（CSS keyframe plantFloat）
- 每個階段植物要明顯比上一階段「更大、更豐盛」

### Gisele 蘋果樹 5 階段

| 階段 | 視覺描述 |
|------|---------|
| 1 🌱 | 一顆棕色種子，一根嫩綠小芽破土而出 |

| 3 🌳 | 粗一點的樹幹，茂密圓形樹冠，葉子飽滿 |
| 4 🌼 | 大樹上出現白色小花（蘋果花），春意盎然 |
| 5 🍎 | 大樹，樹上掛著 6–8 顆紅蘋果，豐收感 |

### Fiona 橘子樹 5 階段

| 階段 | 視覺描述 |

## 動畫規格

### 1. 植物生長動畫（每次澆水/施肥後）
```css
@keyframes plantFloat {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-8px); }
}

@keyframes growBounce {
  0% { transform: scale(1); }
  50% { transform: scale(1.15); }
  100% { transform: scale(1); }
}
```

### 2. 慶祝效果（完成動作後觸發）
- 畫面出現 20–30 個飛散粒子
- Gisele：🍎 ⭐ 🌟 紅色小圓
- Fiona：🍊 ⭐ 🌟 橘色小圓
- 持續時間：2 秒後漸漸消失

### 3. 零用錢數字動畫
```css
/* 數字增加（賺入）：彈跳 + 金色閃光 */
@keyframes moneyPop {
  0% { transform: scale(1); }
  40% { transform: scale(1.3); color: #F5C842; }
  100% { transform: scale(1); }
}

/* 數字減少（提領）：輕微縮小 + 橘紅色 */
@keyframes moneyDrop {
  0% { transform: scale(1); }
  40% { transform: scale(0.85); color: #D94F3D; }
  100% { transform: scale(1); }
}
```

### 4. 按鈕 Hover/Active
- Hover：scale(1.05) + 陰影加深
- Active：scale(0.95)
- Transition：150ms ease

### 成鳥行為設計

#### 成鳥 A：餵食鳥
```
位置：鳥巢左側樹枝（固定）
行為：每 4 秒執行一次啄食
  嘴含小蟲（橘色短線）→ 低頭伸向巢 → 抬頭 → 重複
動畫：birdFeed，1.5s/cycle
```

#### 成鳥 B：守護鳥
```
位置：樹右側空間巡邏
行為：左右來回飛行，弧形軌跡，邊緣 scaleX(-1) 換向
  每 8–12 秒隨機停棲樹枝 2–3 秒，停棲時 idle bob
動畫：birdFly 6s loop + birdFlap 0.3s loop
```

#### 成鳥 C：歌唱鳥
```
位置：樹頂較高枝條（固定停棲）
行為：
  - 停棲時身體輕微搖擺（idleBob）
  - 每 6 秒張嘴一次，飄出 2–3 個音符（♪ ♫，淡出動畫）
  - 每 20 秒飛走（向右飛出畫面），5–8 秒後從左側飛回
動畫：birdSing，musicNote fadeUp 1s
```


## 樹下動物視覺設計

### 出現動畫
每隻動物初次出現：從樹後「探頭→滑入」動畫（0.6s ease-out）

### Gisele 的狐獴 SVG 設計
```
顏色：
  身體：#D4A96A（沙棕）
  腹部：#F5E0C0（淡米）
  臉部標記：#8B6343（深棕眼框）
  眼睛：黑色 + 白色高光

形態：直立站姿（狐獴特徵）
  - 細長身體、小圓耳、長尾
  - 兩隻前爪抱在胸前（警戒姿）

動畫循環（各隻錯開 offset）：
  idle：輕微頭部左右轉（lookAround，3s loop）
  run：快速左右移動（birdFly 概念，水平位移，2s）
  dig：蹲下挖地，土塊小粒子飛出（dig，1.5s）
  每隻動物 8–15s 內隨機執行其中一個行為
```

### Fiona 的兔子 SVG 設計
```
顏色：
  身體：#F5F0E8（乳白）
  耳內：#FFB6C1（粉紅）
  眼睛：#FF6B9D（粉紅大眼）
  鼻子：#FF9BB0（淡粉）

形態：
  - 圓滾滾身體、長耳朵（豎立/收起切換）
  - 短短前爪，蓬鬆尾巴

動畫循環：
  idle：耳朵輕微抖動（earTwitch，2s loop）
  hop：原地跳躍，身體拉伸→壓縮（hop，0.8s，每 3s 一次）
  move：左右蹦跳移動（hopMove，1.2s per hop）
  alert：耳朵完全豎起，靜止 2 秒（alertStand）
```

### 樹下空間佈局
```
樹幹底部往左右各 160px 為動物活動區
動物在 z-index 上低於樹幹，形成「在樹前跑動」的視覺
5 隻同時在場時，位置稍微分散，避免重疊
```

## 提領動畫：猴子拔果實

### 猴子 SVG 設計
```
顏色：
  身體：#A0734A（暖棕）
  臉/手：#D4956A（淺棕膚色）
  眼睛：黑色 + 白高光
  嘴巴：咧嘴大笑

尺寸：約 100px 高

動作分解（timeline，共約 3 秒）：
  0.0s：從畫面右側或左側「盪入」（swing on vine）
  0.8s：停在樹幹旁，伸手夠向最近果實
  1.5s：摘下果實，果實從樹上消失（該位置留下空缺 0.3s 後補回）
  2.0s：猴子舉起果實，露出大笑表情
  2.5s：咬一口果實，出現 😋 表情泡泡
  3.0s：猴子盪回畫面外消失

視覺補充：
  - 猴子身上有一條藤蔓（vine），進出場靠藤蔓盪動
  - 果實被摘走的瞬間，畫面出現「-XXX 元 💸」的飄字
  - 提領完成後，零用錢數字以 moneyDrop 動畫更新
```

## 卡片/元件設計細節

### 角色選擇卡片
- 圓角：24px，陰影：`0 8px 24px rgba(0,0,0,0.12)`
- 按壓：`active:scale-95`
- 內容：大植物 emoji + 孩子名字 + 「的花園」+ 目前累積金額預覽

### 零用錢顯示區
- 背景：圓角黃色卡片（#FFF8DC）
- 累積總額：Nunito 900，56px，深棕色（大字）
- 可提領餘額：Nunito 700，28px，葉綠色（次要）
- 提領按鈕：小型橘色圓角按鈕，「💸 提領」
- 兩列並排：上方累積，下方可提領 + 按鈕

### 提領模態（WithdrawModal）
```
半透明深色遮罩
白色圓角卡片（圓角 20px）

內容：
  🐒 要讓猴子拔幾顆果實？
  可提領餘額：XXX 元
  [輸入框：數字鍵盤]
  [取消]  [確認提領]

輸入驗證：
  - 不可為 0 或負數
  - 不可超過 availableMoney
  - 超出時輸入框變紅 + 提示文字
```

### 生長進度條
- 背景：`--color-cream`，圓角全滿
- 填充：角色主題色，從左到右
- 右側文字：「還需 X 次就能長大！」
- 最高階：「🏆 已達最高成就！」

### 行動按鈕（澆水/施肥）
- 尺寸：140×140px 正圓形
- 大 emoji（48px）+ 中文文字（16px）
- 陰影：`0 6px 20px` 帶色陰影
- 兩個並排，間距 20px
